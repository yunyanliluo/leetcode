# 格雷编码

背答案题

关键是搞清楚格雷编码的生成过程, G(i) = i ^ (i/2);

如 n = 3: 

G(0) = 000, 

G(1) = 1 ^ 0 = 001 ^ 000 = 001

G(2) = 2 ^ 1 = 010 ^ 001 = 011 

G(3) = 3 ^ 1 = 011 ^ 001 = 010

G(4) = 4 ^ 2 = 100 ^ 010 = 110

G(5) = 5 ^ 2 = 101 ^ 010 = 111

G(6) = 6 ^ 3 = 110 ^ 011 = 101

G(7) = 7 ^ 3 = 111 ^ 011 = 100

## 公式题

最开始写的i/2，很慢，改成i>>1，快了很多，速度提升一倍

C++ 
```
class Solution {
public:
    vector<int> grayCode(int n) {
        if(n==0) return {0};
        vector<int> res(1, 0);
        int k=pow(2, n);
        for(int i=1;i<k;++i) {
            res.push_back(i^(i>>1));
        }
        return res;
    }
};
```
